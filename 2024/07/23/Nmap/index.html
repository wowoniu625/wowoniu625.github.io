Nmap

发表于2024-07-23|更新于2024-07-26
|阅读量:

一、 介绍

Nmap是强大的网络扫描器，功能有：

  1. 主机发现
  2. 端口扫描
  3. 服务识别：识别主机上运行的服务和程序版本号
  4. 操作系统检测：检测操作系统的类型和版本

二、 使用Nmap

2.1 添加目标

支持多种添加主机的方式

  1. 单个目标：nmap ip
  2. 多个目标：
    1. 空格分隔：nmap ip1 ip2
    2. IP范围：nmap 192.168.1.2-50
    3. 扫描网段：nmap 192.168.1.0/24
    4. 从文件中读取IP地址：nmap -iL ip.txt，-iL（Input List）
